# Riaz Orthopaedic Hospital Website

A modern, high-performance, single-page informational website for Riaz Orthopaedic Hospital. Designed to be fully responsive and accessible, the site highlights the hospital's specialized services, expert doctors, and contact information with a premium UI.

## 🚀 Tech Stack

*   **Framework:** [React](https://reactjs.org/) bootstrapped with [Vite](https://vitejs.dev/)
*   **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Custom theme variables)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/) (Scroll-triggered variants)
*   **Icons:** [Lucide React](https://lucide.dev/)

## ✨ Key Features

*   **Responsive Design:** Flawless layout scaling from mobile screens to 4K desktop displays.
*   **Animated Mobile Navigation:** Framer Motion-powered hamburger menu with staggered child animations.
*   **Smooth Scrolling:** Built-in anchor routing for single-page navigation (`#services`, `#contact`, etc.).
*   **Scroll-Triggered Animations:** Grid elements use staggered parent-child variant patterns to reveal content dynamically without UI blinking.
*   **Mobile-Optimized Contact Actions:** One-tap `tel:` and `mailto:` links, alongside a fully responsive, custom-styled Google Maps iframe.
